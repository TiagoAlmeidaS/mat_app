# MatApp

Um aplicativo matemático desenvolvido em Flutter para cálculo e visualização de números primos.

## 🎯 Objetivo

O MatApp foi desenvolvido para ser uma ferramenta educativa e prática que auxilia no cálculo e visualização de números primos. É ideal para estudantes e entusiastas de matemática que desejam explorar e entender melhor os números primos.

## ✨ Funcionalidades

- Cálculo de números primos em um intervalo definido
- Interface intuitiva e fácil de usar
- Processamento otimizado para diferentes plataformas
- Visualização clara dos resultados

## 🚀 Como Usar

1. Abra o aplicativo
2. Insira o intervalo desejado (exemplo: 1 a 100)
3. Clique em "Calcular"
4. Visualize os números primos encontrados

## 🛠️ Tecnologias

- Flutter/Dart
- Modular
- Isolates (para processamento paralelo)
